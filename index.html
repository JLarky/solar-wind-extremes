Магнитосфера при экстремальных условиях в солнечном ветре

Данные

Параметры солнечного ветра

В работе использовались данные OMNI дополненные параметрами модели TS05 c 1995 по 2009 год, усреднённые за 5 минут, файлы с которыми выложенные на сайте geo.phys.spbu.ru.

Экстремальные значения параметров

По данным солнечного ветра были построены гистограммы отражающие количество данных из всего набора принимающих значение в диапазоне значений.

  • Распределение значений Bz компоненты межпланетного поля
  • Распределение значений индекса SYM-H
  • Распределение значений давления плазмы
  • Распределение значений скорости солнечного ветра
  • Распределение значений By компоненты межпланетного поля
  • Распределение значений плотности плазмы
  • Распределение значений параметра W1 модели TS05
  • Распределение значений параметра W2 модели TS05
  • Распределение значений параметра W3 модели TS05
  • Распределение значений параметра W4 модели TS05
  • Распределение значений параметра W5 модели TS05
  • Распределение значений параметра W6 модели TS05
